# 📱 PhoneGet PRO

**PhoneGet PRO** — это захватывающий браузерный симулятор перепродажи смартфонов, майнинга и удачи! Пройди путь от владельца кнопочного телефона до технологического магната.

## 🚀 Как запустить игру
Игра опубликована и доступна в интернете:
👉 **[Играть в PhoneGet PRO](https://str1ke546.github.io/phoneget/)**

## 🎮 Основные механики
* **⛏️ Майнинг**: Тапай по главному телефону, чтобы зарабатывать первые деньги.
* **🛒 Авито**: Покупай улучшения и фермы, чтобы рос твой пассивный доход.
* **🎰 Кейсы**: Испытай удачу! Открывай кейсы с шансом выпадения обычных, редких и легендарных девайсов.
* **📦 Склад**: Управляй своей коллекцией и смотри, сколько дохода приносит каждая ферма.

# 📱 PhoneGet PRO (Eng)

**PhoneGet PRO** is an exciting browser-based simulator of smartphone reselling, mining, and luck! Rise from a basic flip phone owner to a tech tycoon.

## 🚀 How to Launch the Game
The game is published and available on the internet:
👉 **[Play PhoneGet PRO](https://str1ke546.github.io/phoneget/)**

## 🎮 Core Mechanics
* **⛏️ Mining**: Tap the main phone to earn your first capital.
* **🛒 Avito (Market)**: Buy upgrades and farms to increase your passive income.
* **🎰 Cases**: Test your luck! Open cases with a chance to drop common, rare, and legendary devices.
* **📦 Inventory**: Manage your collection and see how much income each farm generates.

## 🛠️ Technologies
The project was created using:
* **HTML5** (game structure)
* **CSS3** (modern interface and animations)
* **JavaScript** (economy logic and item drop mechanics)
* **Cursor AI** (development assistance)

## 📝 Development
The project is under active development. Plans include:
- [ ] Adding a player level system.
- [ ] "Spark" effects and critical clicks.
- [ ] PWA manifest for installation on a phone.
